Drugs

The PDBe RelLig pipeline identifies drug-like ligands in the PDB by utilizing drug-target annotations found in the ligand CIF files available from the PDBe FTP. These CIF files are updated weekly and include drug-target data sourced from the DrugBank database.

Annotation process

The PDBe RelLig pipeline parses the input PDB ligand CIF file to retrieve any available drug-target annotations. It then uses the PDBe API to gather all proteins interacting with the ligand. If any of these proteins are pharmacologically active targets, the ligand is annotated as drug-like.
